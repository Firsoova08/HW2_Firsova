 ОСНОВНЫЕ КОМАНДЫ
Инициализация и статус
git init                    # Инициализировать новый репозиторий
git status                  # Показать состояние репозитория
git log --oneline           # Показать историю коммитов (кратко)

Работа с файлами
git add <файл>             # Добавить файл в staging area
git add .                  # Добавить ВСЕ изменения
git commit -m "сообщение"  # Создать коммит с сообщением
touch <имя_файла>          # Создать пустой файл (терминал)
echo "текст" > файл        # Создать файл с текстом (терминал)

Ветки (Branches),
git branch                  # Показать все ветки
git checkout -b <имя>      # Создать и перейти в новую ветку
git checkout <имя>         # Переключиться на ветку
git branch -a              # Показать все ветки (включая удаленные)

 РАБОТА С УДАЛЕННЫМ РЕПОЗИТОРИЕМ (GitHub/GitFlic),
Подключение
git remote add origin <URL>  # Добавить удаленный репозиторий
git remote -v                 # Показать подключенные репозитории

Отправка и получение
git push -u origin main     # Отправить изменения в ветку main
git push -u origin feature  # Отправить ветку feature на сервер
git pull                    # Получить изменения с сервера
Создание ветки feature:
git checkout -b feature,

Подключение к GitHub:
git remote add origin https://github.com/[мой-логин]/git-practice-[Фамилия].git
